# Queue Simulator (MM1, MG1, Multi-server)

## Description

This project implements queueing models:

* MM1
* MG1
* Multi-server

Using C# with theoretical calculations and discrete event simulation.

## Features

* Input λ (arrival rate) and μ (service rate)
* Compute performance measures
* Run simulation
* Compare simulation with theory

## How to Run

### Method 1 (Recommended)

1. Open terminal in project folder
2. Run:
   dotnet run

### Method 2

Run the executable from:
Executable/MM1QueueSimulator.exe
